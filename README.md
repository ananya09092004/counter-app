Simple Counter App
====================
This is a basic counter application built with HTML, CSS, and JavaScript. The app allows users to increment, decrement, and reset a counter.

Description
-----------
The application consists of a simple interface with buttons to control the counter. The current count is displayed on the screen. The app is fully functional and can be used as a starting point for more complex projects.

Usage
-----
1. Open the index.html file in a web browser to launch the application.
2. Click the "+" button to increment the counter.
3. Click the "-" button to decrement the counter.
4. Click the "Reset" button to reset the counter to zero.

Project Structure
----------------
The project consists of the following files:
- index.html: The main entry point of the application.
- styles.css: Styles and layout for the application.
- script.js: JavaScript code for the counter functionality.
- package.json: Project dependencies and scripts.

Requirements
------------
- A modern web browser to run the application.
- A code editor or IDE to modify the code.

Installation
------------
No installation is required. Simply open the index.html file in a web browser to run the application.
